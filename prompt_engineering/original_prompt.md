I want to create "bit:" a bash interpreter translator. This is a CLI tool that works as follows: it is invoked in a linux shell via the CLI command "bit <instruction" and converts <instruction> into the relevant linux command. So for instance "bit create folder new_folder" should be translated to "mkdir new_folder." This latter command should appear immediately in the command line, ready to be submitted by pressing enter. The translation is carried out via LLM: bit should use a local OLLAMA installation, and the configuration (including the model name to be used) should be stored into file ~/.bit. The relevant OLLAMA model can be selected among the installed ones via command bit --setup. The CLI should also feature a bit --help feature. Implement this in Python, and develop suitable toml, MANIFEST, README.md etc. It should be possible to build a distribution via python -m build and it should be possible to inoke the tool, after being installed via pip, by simply typing bit.
